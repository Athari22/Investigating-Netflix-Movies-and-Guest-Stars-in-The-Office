## General Info
This project is part of the **Data Scientist with Python** career track on [DataCamp](https://app.datacamp.com/learn/career-tracks/data-scientist-with-python?version=5).  
It focuses on exploring **Netflix movies** and **guest stars in *The Office*** using Python, pandas, and matplotlib.

---

## Description
This project applies Python programming skills to solve real-world data science problems:  

- Analyze whether **Netflix movies are getting shorter over time**.  
- Identify **guest stars that appear in the most popular episodes of *The Office***.  
- Practice Python fundamentals, including **lists, loops, pandas, and matplotlib**.
